# Create Account Testing Project

This project is a simple Java application developed using Maven.

## Description

The application simulates a basic account creation system with input validation, including:

* First name and last name validation
* Email format checking
* Password length validation
* Password confirmation matching

## Testing

Unit tests were implemented using JUnit to verify different scenarios, including:

* Valid account creation
* Invalid email format
* Short password
* Password mismatch
* Missing fields

Total test cases: 15
All tests passed successfully.

## Technologies Used

* Java
* Maven
* JUnit
